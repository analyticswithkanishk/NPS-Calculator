## 🚀 Project Overview

This project is a simple yet practical implementation of the **Net Promoter Score (NPS)** using Python. NPS is a widely used business metric that helps organizations measure **customer satisfaction and loyalty**.

Companies like Apple, Amazon, and Netflix rely on NPS to understand how customers perceive their products and services.

> 📌 **Formula:**  
> **NPS = % Promoters - % Detractors**

---

## 📊 Customer Segmentation

Customers are classified into three categories based on their ratings:

| Category     | Score Range | Description           |
|--------------|------------|-----------------------|
| Promoters    | 9 - 10     | Loyal & happy users   |
| Passives     | 7 - 8      | Neutral users         |
| Detractors   | 0 - 6      | Unhappy users         |

---

## ⚙️ What This Project Does

✔️ Generates **1000 random customer ratings (0–10)** using NumPy  
✔️ Classifies customers into **Promoters, Passives, Detractors**  
✔️ Calculates percentage distribution  
✔️ Computes **Net Promoter Score (NPS)**  
✔️ Provides **business insight using conditional logic**  
✔️ Performs basic **Exploratory Data Analysis (EDA)**
